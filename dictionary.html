📚 Dictionary Converter

Converts a dictionary into the reader's binary lookup format (.idx + .dat), including the .spx sidecar that makes on-device lookups fast.

1. Choose a dictionary file

Recommended sources: Jitendex (vocabulary, Yomitan format), JMnedict in Yomitan format (names), or a grammar reference in Yomitan format. MDict (.mdx) files need the desktop Python tool.

2. Which dictionary slot is this?

Large dictionaries (Jitendex is ~60 MB) take a minute and some memory to convert — everything stays on your device.

Install on the device

The download is a zip containing a dict/ folder. Unzip it at the SD card root (or upload the files into /dict/ via the reader's Wi-Fi web file transfer). After installing all three dictionaries your card looks like:

/dict/
  jmdict.idx  jmdict.dat  jmdict.spx      (vocabulary — required)
  jmnedict.idx  jmnedict.dat  jmnedict.spx  (names — optional)
  grammar.idx  grammar.dat  grammar.spx     (grammar — optional)
Output matches tools/dict_convert/convert_jmdict.py and scripts/gen_dict_spx.py from the matcha-reader firmware byte-for-byte.

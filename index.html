🍵 Matcha Reader Tools

Manga Dictionary Fonts

Prepare content for your Matcha Reader

Everything runs in your browser — including on a smartphone. Your files never leave your device, except manga panel images sent to Google's Gemini API for text recognition when you provide your own API key.

📖

Manga Converter

Turn a CBZ, EPUB, PDF, or a folder of page images into the reader's manga format — panel detection, Japanese OCR, and English translations included.
📚

Dictionary Converter

Convert Yomitan dictionaries (Jitendex, JMnedict, grammar references) or jmdict-simplified JSON into the on-device lookup format.
🔤

Font Converter

Convert TTF/OTF fonts (like Japanese reading fonts) into .cpfont files for the SD card, with kerning and ligatures.

Getting files onto the device

Each tool downloads a zip laid out for the SD card. Either unzip it onto the card with a card reader, or — with no computer at all — use the reader's built-in Wi-Fi web file transfer to upload the files straight from your phone.

Companion tools for the Matcha Reader firmware (a CrossPoint Reader fork with Japanese learning features). Output formats match the firmware's Python conversion tools byte-for-byte. Open source, MIT licensed.

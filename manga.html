📖 Manga Converter

Converts manga into the reader's panel format: detects panel rectangles, asks Gemini what text appears in each panel (plus an English translation), and packs everything into the binary format the device reads natively.

OCR sends each panel image to the Gemini API and can take a while for a full volume — keep this tab open. The screen is kept awake during conversion. On the free API tier, rate limits may make long volumes slow; the converter retries automatically.

Install on the device

The download is a zip with one folder (named after the title). Put that folder anywhere on the SD card — /manga/YourTitle/ is a nice convention — either with a card reader or via the reader's Wi-Fi web file transfer. The device finds any folder containing panels.idx automatically, at any depth.

Output matches tools/manga_convert/convert_manga.py from the matcha-reader firmware — including its YOLO panel detector (in-browser via ONNX Runtime) and PDF input (in-browser via PDF.js).
